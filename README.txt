Funcionamento:
A tela inicial apresenta os botões para cadastrar, alterar e remover projetos, além de uma lista de projetos ja cadastrados.
Ao clicar na linha de um dos projetos já cadastrados aparecerão os campos de cadastro, alteração e remoção de atividades, além de uma lista de atividades já cadastradas.
As ações serão sempre executadas no projeto previamente selecionado na tabela.
Para cadastrar e alterar atividades/projetos basta preencher os campos e pressionar o boão de finalizar.
Para remover atividades/projetos é necessario somente digitar o ID da mesma, removendo um projeto você tambem remove todas as atividades ligadas a ele.

As ferramentas utilizadas nesse projeto foram:
- Eclipse IDE
- PgAdmin4
